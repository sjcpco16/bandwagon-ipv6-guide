## 在 KiwiVM 面板里启用 IPv6：完整步骤

搬瓦工把 IPv6 的激活操作做到了面板里，步骤不复杂：

1. **登录 KiwiVM 控制面板**，进入你的 VPS 管理页面

2. **点击左侧菜单的"IPv6 network"**，进入 IPv6 管理界面

3. **点击"Activate /64 IPv6 subnet"**按钮，等待面板提示激活成功

4. **在面板里重启 VPS**，让网络配置生效

5. 重启完成后回到"IPv6 network"页面，确认分配到的 /64 地址段

整个过程基本就是点三下。激活完重启就能用，不需要手动配置网络接口文件。

问题来了：如果你在第三步看到的提示是"IPv6 is not available at this location"怎么办？

说明你当前的机房不在上面那张支持列表里。解决办法是进 KiwiVM 的机房迁移功能，迁到一个支持 IPv6 的机房就行了。迁移过程会消耗一部分流量，建议先清理一下 VPS 里不需要的文件再迁。

## IPv6 激活之后能用来做什么

说一个实际场景：有些境外服务或 API 需要 IPv6 才能访问，或者你想给自己的站点配一个 AAAA 记录，让支持 IPv6 的访客走双栈。搬瓦工激活的是 /64 的子网，地址多到根本用不完，拿来跑容器、分配给不同服务都没问题。

另外有一点要注意：如果你切换机房，IPv6 地址会保持不变，但换机房后建议先把 IPv6 删掉再重新激活一次，否则可能会出现延迟偏高的情况。
